S2TW WORKBENCH

把簡中酒館內容轉成繁中玩家能安心使用的格式。

這個倉庫專門處理 SillyTavern 預設、世界書與角色卡。每個工具都先拆區塊預覽，再讓你編輯文字與匯出 JSON，避免整份檔案盲轉後才發現格式壞掉。

安全轉換規則

OpenCC 負責簡繁字形，詞庫只處理保守的台灣用語；容易誤傷 JSON、英文縮寫、程式碼或美化規則的詞條已移除。

01正則 findRegex 與 replaceString 預設不做簡繁替換。
02酒館助手 JavaScript 只給你預覽與編輯，不自動轉換程式碼內容。
03角色卡與世界書會保留原本 JSON 結構，只改安全文字欄位。
